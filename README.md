# RL-AStock-Trader V23

基于强化学习(PPO)+9颗子弹动态管理的A股量化交易策略 **V23优化版**。

## 🎯 版本特性

| 参数 | 值 | 说明 |
|------|-----|------|
| **子弹总数** | 9颗封顶 | 最大90%仓位 |
| **单颗仓位** | 10% | 每颗占资金量的10% |
| **单次加减仓** | 最多3颗 | 动作1-3加、4-6减 |
| **最大持仓天数** | 15天 | 缩短周期，增加交易频次 |
| **动态止损** | 5% + ATR追踪 | 自适应止损机制 |
| **冷静期** | 已取消 | 提高交易灵活性 |

## 📊 V23核心改进

### 1. 奖励函数优化
- **降低满仓惩罚**：从4.0降至1.5，鼓励积极建仓
- **增强小额盈利奖励**：1-5%盈利即给予奖励，提高胜率导向
- **优化持仓周期**：最佳持仓1-4天，鼓励短线交易
- **增强趋势对齐奖励**：多头排列额外加分

### 2. 止损机制增强
- **动态ATR止损**：根据波动率自适应调整止损幅度
- **追踪止损**：盈利超过3%后启用，锁定利润
- **降低止损线**：从6%降至5%，减少不必要止损
- **降低止损惩罚**：从30降至15，减少心理压力

### 3. 交易频次提升
- **缩短持仓周期**：从20天降至15天
- **放宽开仓限制**：首次建仓从5颗增加到7颗
- **降低频繁交易惩罚**：从0.3降至0.1
- **降低手续费**：从0.05降至0.03
