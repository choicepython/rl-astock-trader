# RL-AStock-Trader V22

基于强化学习(PPO)+9颗子弹动态管理的A股量化交易策略 **V22激进版**。

## 🎯 版本特性

| 参数 | 值 | 说明 |
|------|-----|------|
| **子弹总数** | 9颗封顶 | 最大90%仓位 |
| **单颗仓位** | 10% | 每颗占资金量的10% |
| **单次加减仓** | 最多3颗 | 动作1-3加、4-6减 |
| **最大持仓天数** | 20天 | 超时强制清仓 |
| **动态止损** | 6% | 浮亏超6%立即止损 |
| **冷静期** | 5天 | 止损后禁止加仓 |

## 📊 V22核心改进

1. **6%动态止损** - 最大回撤控制在8%以内
2. **5天冷静期** - 止损后防止频繁加仓
3. **让利润奔跑** - 持仓成本惩罚降低

## 🚀 快速开始

### 1. 训练模型

```python
from train_rl_strategy import train_rl_model
train_rl_model("002156")  # 输入股票代码
```

### 2. 策略回测

```python
from indicator_strategy import IndicatorStrategy

strategy = IndicatorStrategy(symbol="002156")
strategy.backtest_rl("ppo_trading_002156")
```
