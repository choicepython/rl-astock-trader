# RL 强化学习量化交易策略

基于 PPO 强化学习算法 + 9颗子弹动态仓位管理系统，专为A股市场设计的量化交易策略。

---

## 📋 快速开始

### 1. 训练强化学习模型

```python
from train_rl_strategy import train_rl_model
model = train_rl_model("002156")  # 输入股票代码
```

或直接运行脚本：
```bash
python train_rl_strategy.py
```

### 2. 策略回测

```python
from indicator_strategy import IndicatorStrategy

strategy = IndicatorStrategy(symbol="002156")
strategy.backtest_rl("ppo_trading_002156")
```

## 🎯 核心参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **子弹总数** | 9颗封顶 | 最大90%仓位 |
| **单颗仓位** | 10% | 每颗占资金量的10% |
| **单次加减仓** | 最多3颗 | 动作1-3加、4-6减 |
| **最大持仓天数** | 15天 | 超时强制清仓 |
| **开仓限制** | ≤7颗 | 首次建仓最多7颗 |
| **动态止损** | 5% | 浮亏超5%立即止损 |
| **追踪止损** | 5% | 盈利超5%后启动追踪 |

## 💰 完整奖励评价体系

### 持仓期每日评分

| 评分项 | 规则 | 分值 | 设计目的 |
|--------|------|------|---------|
| **仓位惩罚** | 满仓(90%) | -0.5 | 控制风险，预留子弹 |
| | 80-90% | -0.2 | 轻度提醒 |
| **持仓成本** | 每持有1天 | -0.002/颗 | 鼓励快进快出 |
| **趋势奖励** | 价格>MA20 | +0.2 | 趋势跟随 |
| | 多头排列 | +0.15 | 确认趋势强度 |
| **利润奖励** | 浮盈增长 | ×4.0 | 让利润奔跑 |
| | 浮亏减少 | ×0.2 | 鼓励止损 |
| **大涨奖励** | 2-5% | +(涨幅-2)×2.5 | 捕捉趋势 |
| | 5-8% | +4.0 | 强化持仓 |
| | 8%+ | +10.0 | 超级奖励 |
| **回撤惩罚** | 回撤>5% | 回撤^1.2×0.15 | 非线性惩罚 |
| **持仓天数** | 1-4天 | +0.5 | 最佳周期 |
| | 5-8天 | +0.25 | 理想周期 |
| | 9-12天 | +0.1 | 正常周期 |
| | 13-15天 | -0.15/天 | 惩罚过长 |

## 📊 观测空间 (22维特征)

### 技术指标 (14维)

| 指标 | 说明 | 用途 |
|------|------|------|

## 🎯 策略设计理念

| 目标 | 实现方式 |
|------|---------|
| **鼓励进攻** | 利润奖励4倍、大涨超级奖励(8%+10分) |
| **控制风险** | 满仓轻罚(-0.5分)、开仓限制7颗 |
| **趋势跟随** | MA20奖励、连续盈利加分 |
| **让利润跑** | 追踪止损5%、大涨奖励递增 |
| **减少频繁交易** | 频繁操作惩罚 |
