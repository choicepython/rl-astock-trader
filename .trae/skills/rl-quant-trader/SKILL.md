# RL 强化学习量化交易策略 (V11稳定版)

基于 PPO 强化学习算法 + 9颗子弹动态仓位管理系统，专为A股市场设计的稳定收益策略。

---

## 📋 快速开始

### 1. 训练强化学习模型

```python
from train_rl_strategy import train_rl_model
model = train_rl_model("002156")  # 输入股票代码
```

或直接运行脚本：
```bash
python train_rl_strategy.py
```

### 2. 策略回测

```python
from indicator_strategy import IndicatorStrategy

strategy = IndicatorStrategy(symbol="002156")
strategy.backtest_rl("ppo_trading_002156")
```

## 🎯 核心参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **子弹总数** | 9颗封顶 | 最大90%仓位 |
| **单颗仓位** | 10% | 每颗占资金量的10% |
| **单次加减仓** | 最多3颗 | 动作1-3加、4-6减 |
| **最大持仓天数** | 22天 | 超时强制清仓 |
| **开仓限制** | ≤5颗 | 首次建仓最多5颗 |

## 💰 V11完整奖励评价体系

### 持仓期每日评分

| 评分项 | 规则 | 分值 |
|--------|------|------|
| **A.仓位惩罚** | 满仓(100%) | -4.0分 |
| | 90%-99% | -0.8分 |
| | 90%以下 | 0分 |
| **B.持仓成本** | 每持有1天 | -0.005分/颗 |
| **C.趋势奖励** | 价格>MA20 | +0.15分 |
| **D.利润奖励** | 浮盈增长 | ×2.5倍 |
| | 浮亏减少 | ×0.5倍 |
| | 连续盈利≥2天 | +0.3分/天 |
| **E.大涨奖励** | 3%-5%涨幅 | +(涨幅-3%)×1.0 |
| | 5%-8%涨幅 | 额外+2.0分 |
| | 8%以上 | 额外+5.0分 |
| **F.回撤惩罚** | 回撤>8%后 | -回撤%×0.1 |

### 动作执行评分

| 评分项 | 规则 | 分值 |
|--------|------|------|
| **G.频繁惩罚** | 单日连续交易 | -0.3分 |
| **H.逆势惩罚** | 浮亏时加仓 | -0.5分/颗 |
| **I.开仓限制** | 首次>5颗 | -2.0分 |

### 清仓评分

| 结果 | 奖励 |
|------|------|
| 盈利 | +5分 + 盈利比例奖励 |
| 亏损 | -5分 + 亏损比例惩罚 |

## 📊 观测空间 (20维特征)

### 技术指标 (14维)

| 指标 | 说明 |
|------|------|
| dif, dea, macd | MACD指标 |
| k, d | KD随机指标 |
| rsi | 相对强弱指数 |
| MA5, MA10, MA20, MA60 | 均线系统 |
| vol_ratio | 量比 |
| boll_width | 布林带宽 |
| atr | 真实波幅 |
| price_ma20_ratio | 价格相对均线比 |

### 仓位状态 (6维)

| 特征 | 说明 |
|------|------|
| position_count_norm | 归一化持仓数 |
| floating_pnl_pct | 当前浮盈/浮亏率 |
| hold_days_norm | 持仓天数比例 |
| avg_cost_price_ratio | 成本价差率 |
| drawdown_since_entry | 入场后回撤 |
| profit_run_days | 连续盈利天数 |

## 🎯 策略设计理念

| 目标 | 实现方式 |
|------|---------|
| **鼓励进攻** | 利润奖励2.5倍、大涨超级奖励(8%+5分) |
| **控制风险** | 满仓重罚(-4分)、开仓限制5颗 |
| **趋势跟随** | MA20奖励、连续盈利加分 |
| **让利润跑** | 回撤容忍8%、大涨奖励递增 |
| **减少频繁交易** | 频繁操作惩罚 |
